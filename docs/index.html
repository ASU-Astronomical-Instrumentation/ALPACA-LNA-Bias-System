Intro

The code from this repository was developed using a python virtual environment located in the root home directory. The packages have since been installed outside of an environment. As a result, the user should not have to use the venv. It is however, still recommended.

On import/run, the ALPACABias.py module automatically tries to initialize all of the boards in the system. During this process all of the bias outputs are set to their minimum.

For details on usage or documentation, please see the API.

Please note that this module requires root access.
